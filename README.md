# Healios

An application that allows buyers and sellers to trade medical equipment. 

Buyers -> can view products, make orders, track them, make payments, and create product reviews

Sellers -> Add products, manage the stock, process orders, and reply to reviews.

Application offers analytics for sellers. 

Application can be adapted for whatever type of product and product categories.
